STEP 01

번호 유형 선택 후 입력

보유하신 화물 번호 유형에 맞게 화물관리번호, B/L번호, 마스터 B/L번호 중 하나를 선택합니다. 입력 전 번호 형식과 허용 문자를 자동으로 검증하여 오입력을 방지합니다.

화물관리번호 예시

KRPUS2409XXXXXXXX

STEP 02

서버를 통한 안전한 실시간 조회

입력값이 서버 측 API 프록시를 거쳐 관세청 Unipass에 안전하게 전달됩니다. API 키는 서버에 보관되어 외부에 노출되지 않으며, 결과를 즉시 응답합니다.

입력값 유효성 검증 (길이·문자 체계)
서버 측 API 키 보안 관리
60초 캐시로 중복 호출 방지
STEP 03

전체 통관 현황 한 화면에서 확인

화물명·진행률·항만 정보와 함께 시간순 통관 타임라인을 한 화면에 표시합니다. 각 단계별 처리 일시와 신고번호까지 상세하게 제공됩니다.

8단계 전부 추적

한국 수입 화물의 통관 처리 과정 전체를 단계별로 시각화합니다. 현재 어느 단계에 있는지, 언제 처리되었는지 정확하게 파악하세요.

최신 상태가 현재 상태 배지와 함께 상단에 표시됩니다.

각 단계별 신고번호(신고접수 이후)도 함께 제공됩니다.

반송·취하 처리된 화물도 명확한 상태 표시로 안내합니다.

신고전

Pre-Declaration

수입 신고가 접수되기 전 대기 상태입니다. 화물이 입항하여 신고 처리를 기다리고 있습니다.

신고접수

Declaration Received

수입 신고서가 세관에 접수된 상태입니다. 신고번호가 부여되어 공식적인 심사가 시작됩니다.

검사진행

Physical Inspection

세관 담당관이 실물 화물을 직접 검사 중인 상태입니다. X-Ray 검사 또는 현장 개장 검사가 진행될 수 있습니다.

심사진행

Under Examination

신고 서류 및 세율·원산지 등을 세관 심사관이 검토하는 단계입니다. 서류 보완 요청이 발생할 수 있습니다.

납세대기

Awaiting Duty Payment

관세 및 부가세 납부 완료를 기다리는 상태입니다. 납부 확인 후 즉시 통관 완료 처리됩니다.

통관완료

Customs Cleared

모든 통관 절차가 완료된 상태입니다. 화물이 정상적으로 반출되어 내륙 운송 또는 배송을 준비할 수 있습니다.

반송·취하

Rejected / Withdrawn

신고가 취하되거나 세관에 의해 반송 처리된 상태입니다. 재신고 또는 관세사 상담이 필요합니다.

상태 미확인

Unknown

API에서 상태 정보를 확인할 수 없는 경우입니다. 관세청 Unipass에서 직접 조회하세요.
