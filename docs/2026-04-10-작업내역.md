## 1. PWA 웹앱 초기 구축 (`204b553`)

### 프로젝트 스택
- **Next.js 15** App Router + TypeScript
- **Tailwind CSS** + shadcn/ui 컴포넌트
- **Serwist** 서비스워커 (PWA)
- **관세청 Unipass OPEN API** 연동

### 구현 파일 (50개)

**앱 레이아웃**
- `src/app/layout.tsx` — 루트 레이아웃, PWA 메타태그, safe-area 대응
- `src/app/page.tsx` — 화물 조회 메인 페이지
- `src/app/history/page.tsx` — 검색 기록 페이지
- `src/app/settings/page.tsx` — 설정 페이지
- `src/app/offline/page.tsx` — 오프라인 fallback 페이지

**API**
- `src/app/api/cargo/route.ts` — 서버사이드 관세청 API 프록시 (API 키 보호)
- `src/lib/api.ts` — 클라이언트 fetch 함수 + XML 파싱
- `src/lib/types.ts` — 타입 정의 (`CargoInfo`, `ClearanceProgressItem`, `SearchType` 등)

**컴포넌트**
- `src/components/cargo/cargo-detail-card.tsx` — 화물 상세 카드
- `src/components/cargo/search-form.tsx` — 검색 폼 (3가지 번호 유형)
- `src/components/cargo/status-badge.tsx` — 통관 상태 배지
- `src/components/cargo/status-timeline.tsx` — 통관 진행 타임라인
- `src/components/layout/header.tsx` — 앱 헤더
- `src/components/layout/bottom-nav.tsx` — 하단 탭 네비게이션
- `src/components/install/install-prompt.tsx` — PWA 설치 유도 배너

**PWA**
- `src/sw.ts` — Serwist 서비스워커 (Network-First / Cache-First / StaleWhileRevalidate)
- `public/manifest.json` — PWA 매니페스트

**유틸**
- `src/lib/utils.ts` — `formatDateTime`, `getCurrentClearanceStatus`, `calcProgressPercent`
- `src/hooks/use-cargo.ts` — 화물 조회 커스텀 훅

## 3. 랜딩 페이지 추가 및 앱 전면 개편 (`0d78eb1`)

### 랜딩 페이지 (`public/landing.html`)
- 독립 HTML 파일 (Next.js 빌드 외부)
- `next.config.mjs` rewrite: `/ → /landing.html`
- **디자인**: 다크 zinc-950 + 에메랄드-500 강조색, Solar/Tabler 아이콘
- **섹션 구성**: Hero → Metrics → 이용방법 → 통관단계 → 주요기능(Bento) → PWA → CTA → Footer
- **외부 링크**: 관세청 유니패스, 공공데이터포털, GitHub 실제 URL 적용

### 앱 라우트 구조 변경
```
src/app/
├── layout.tsx          ← 최소화 (html/body만)
├── (app)/
│   ├── layout.tsx      ← 앱 쉘 (Header + BottomNav + PWA 배너)
│   ├── app/page.tsx    ← 화물 조회 (기존 page.tsx 이동)
│   ├── history/        ← 검색 기록
│   ├── settings/       ← 설정
│   └── offline/        ← 오프라인
```

## 4. 랜딩 페이지 개선 (`ed62c70`)

### 앱 링크 준비중 처리
- `/app` 링크 전체에 `data-app-link` 속성 추가
- 클릭 시 페이지 이동 차단 → 하단 글래스 토스트 표시
- 토스트: "아직 준비중입니다." — 2.5초 후 자동 소멸

### 네비게이션 활성화
- `IntersectionObserver` 기반 스크롤 위치 감지
- `visibleSections` Set으로 번쩍임 없이 안정적 활성 섹션 추적
- 활성 메뉴: 에메랄드 반투명 타원형 배경 (레이아웃 이동 없음)
- 메뉴 순서 실제 섹션 순서와 일치: 이용방법 → 통관단계 → 주요기능 → PWA

### 아이콘 수정
- `solar:microscope-bold` (미존재) → `solar:clipboard-check-bold`
- `solar:magnifier-*` (철자 오류) → `solar:magnifer-linear` 통일
- `solar:wifi-off-bold` (미존재) → `solar:cloud-storage-bold`
- 랜딩 로고: `solar:import-bold` → `tabler:package-import` (앱과 동일)
